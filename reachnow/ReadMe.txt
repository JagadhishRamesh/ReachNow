How to run the ReachNow Project

1. Download the  zip file

2. Extract the file and copy reachnow folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4. Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with name reachnow

6. Import reachnow.sql file(given inside the zip package in reachnow folder)

7.Run the script http://localhost/reachnow/login.php (frontend)
